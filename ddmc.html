What is DDMC?

DDMC is a mod launcher for Doki Doki Literature Club. It started as a project for myself, but thought, why not upload it? Oh yeah, read over the shitcode here

Features

  • Profile-based mod launching
  • Playtime tracking
  • Discord Rich Presence
  • Dark & light theme toggle
  • CTRL+L to open Log Viewer

Usage Instructions

  1. Import your vanilla DDLC folder by selecting the game’s original install location.
  2. Unzip your mods before importing. Make sure their folder structure starts at the root of the game (e.g., contains rpa files or folders like game/).
  3. Mods with .rpa files in the root will be auto-organized into the proper /game directory on import.
  4. Create a Profile, select the mod, and set a preferred executable if needed.
  5. Launch and enjoy :3
