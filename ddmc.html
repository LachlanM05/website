What is Doki Doki Modding Club?

DDMC is a mod launcher/manager for Doki Doki Literature Club. It started as a project for myself, but I thought others may find it useful too. Read the Source Code here.

Features

  • Profile-based mod management/launching.
  • Playtime tracking.
  • Discord Rich Presence
  • Dark and Light Mode.
  • Easy access logs with CTRL+L.
